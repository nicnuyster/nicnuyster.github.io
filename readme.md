# 🌐 Персональное портфолио / CV-сайт

Одностраничный адаптивный веб-сайт, предназначенный для представления профессионального опыта, образования, проектов и контактной информации.

---

# 🛠️ 1. Стек технологий

### 💻 Frontend
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

### 🎨 Стилизация
- CSS Flexbox
- CSS Grid
- CSS Variables
- Media Queries
- CSS Transitions & Animations
- Glassmorphism UI

### 🌐 Browser API
- Intersection Observer API
- requestAnimationFrame
- DOM API
- Scroll API

### 🚀 Развертывание
- GitHub Pages

---

# 📖 2. Общее описание проекта

Проект представляет собой современное одностраничное портфолио, выполняющее роль интерактивного онлайн-резюме.

## ✨ Основные разделы

- 👤 О себе
- 🎓 Академический опыт
- 💼 Коммерческий опыт
- 📂 GitHub-проекты
- 📞 Контактная информация

Основной акцент сделан на современном пользовательском интерфейсе, плавных анимациях и высокой производительности.
